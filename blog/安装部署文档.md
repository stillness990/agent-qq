# agent-qq 安装部署文档

## 1. 环境要求

- Ubuntu 24.04
- Docker 与 Docker Compose
- Python 3.12
- NapCat QQ
- 已配置完成的 Claude Code CLI

## 2. 获取项目

```bash
git clone https://github.com/stillness990/agent-qq.git
cd agent-qq
```

如果是本地开发目录，直接进入项目：

```bash
cd /home/ww/agent-qq
```

## 3. 配置环境变量

```bash
cp .env.example .env
nano .env
```

关键配置：

```env
ONEBOT_WS_URL=ws://host.docker.internal:3001
ADMIN_QQ_IDS=你的QQ号
CLAUDE_CONFIG_DIR=/home/ww/.claude
```

## 4. 本地运行

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python bot.py
```

## 5. Docker 部署

```bash
docker compose up -d --build
```

查看日志：

```bash
docker compose logs -f agent-qq
```

## 6. QQ 测试

向机器人 QQ 私聊发送：

```text
/help
/status
/ask 你好
```
